Shamir Secret Sharing Scheme

This tools offers a practical implementation of Shamir's Secret Sharing Scheme (SSSS), a cryptographic method for securely splitting a secret into multiple parts. The secret can only be reconstructed when a sufficient number of parts are combined, ensuring that no single part reveals any information about the original secret.

This implementation is inspired by Ian Coleman's SSSS tool with additional features (e.g. optional encryption, self-contained part files).

How to use: creating a shared secret

  • Navigate to the Split page.
  • Enter the secret you want to split. This can be any text, such as a password, a private key, or any sensitive information.
  • Specify the total number of parts to create and the minimum number of parts required to reconstruct the secret.
  • (Optional) Set a password to encrypt the secret before splitting it. This adds an extra layer of security.
  • Either save each part or download them as self-contained files.

How to use: reconstructing the secret

  • Either navigate to the Combine parts page or open a self-contained part file.
  • Paste the other parts needed to reach the minimum threshold.
  • (If applicable) Enter the password used to encrypt the secret.
